# Uarty
Uarty is a minimal and open source program that lets you read and write using serial communications on Linux systems.

## Usage
`Usage: uarty <port> <baud>`  
The tool needs two parameters which are:  
>**Port**: Specify the serial port to read from (file descriptor)  

>**Baud rate**: Specify the speed of reading and writing. Supported values are: `9600, 19200, 38400, 57600, 115200, 230400`

# AUR
This program is distributed on the AUR repositories for Arch linux under the package name `uarty-git`
